**VN Markdown to Unicode** là một công cụ web tĩnh (Static Web Tool) mạnh mẽ giúp chuyển đổi các định dạng Markdown tiêu chuẩn sang ký tự Unicode giả lập (Mathematical Alphanumeric Symbols). 

Đặc biệt, công cụ được tối ưu hóa riêng cho **Tiếng Việt**, giải quyết triệt để vấn đề lỗi font chữ **đ/Đ** và các dấu phụ mà hầu hết các công cụ quốc tế (như YayText) đang gặp phải.

---

## 🚀 Tính năng nổi bật

- ✅ **Full Markdown Support**: Tự động nhận diện và chuyển đổi `**Bold**`, `*Italic*`, `***Bold-Italic***` và `` `Monospace` ``.
- ✅ **Vietnamese Optimized**: Thuật toán xử lý ký tự thông minh giúp hiển thị chữ **đ/Đ** và các dấu tiếng Việt (NFD/NFC) đồng bộ với font chữ Unicode.
- ✅ **Zero Footprint**: Chạy hoàn toàn trên trình duyệt (Client-side), không gửi dữ liệu về máy chủ, bảo vệ quyền riêng tư tuyệt đối.
- ✅ **Smart HTML Decoding**: Hỗ trợ xử lý các ký tự đặc biệt (ngoặc kép, Hán tự, Emoji) mà không làm hỏng cấu trúc văn bản.

## 🛠 Công nghệ sử dụng

- **JavaScript (Vanilla)**: Xử lý chuỗi và mapping ký tự.
- **Marked.js**: Thư viện parser Markdown hiệu năng cao.
- **HTML5/CSS3**: Giao diện Responsive, tối ưu hóa cho cả desktop và mobile.
- **GitHub Pages**: Nền tảng hosting web tĩnh.

## 📖 Cách sử dụng

1. Truy cập vào trang công cụ: [Link GitHub Pages của bạn tại đây]
2. Nhập nội dung định dạng Markdown vào ô **Input**.
3. Kết quả Unicode sẽ hiển thị ngay lập tức tại ô **Output**.
4. Nhấn **Sao chép** và dán vào Facebook, Zalo, LinkedIn, YouTube... để tạo điểm nhấn cho nội dung của bạn.
